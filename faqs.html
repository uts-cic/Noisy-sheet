NNoisy sheets is an add-on that works in google spreadsheets, which allows users to: add noise to a dataset; share the unique datasets with a student cohort; and calculate the results of functions for the students' versions of the data.

To use the "Noisy sheets" add-on:

  1. This original spreadsheet should be in a folder within your google drive (i.e., it cannot be in the main "My Drive" folder). You need three sheets:
    1. Data - a spreadsheet of data, with column headings, and rows of data. If you wish to protect the spreadsheet from editing, you may do so and this protection will be copied.
    2. Questions - a sheet with two columns: 'Questions' and 'Answer'. The 'Questions' (column A) describe the calculation you are performing or question you will ask. The 'Answers' (column B) perform calculations that refer to the 'Data' sheet; you can use sql formulae or normal formulae here.
    3. Email or Form Responses - see instructions below
  2. You should create a 'Questions'.
  3. There are two modes for using Noisy Sheets.
    1. Recommended, via Google Form: This option distributes copies of the dataset to students who sign up via a Google Form. To use this option, you must setup a Google Form and select your data spreadsheet as the response destination (see instructions here); you must check the box to collect emails.
    2. By class list (not recommended): You can also upload a list with student information, the addon will create a copy of the sheet for each student in the list. The addon is not optimised for this option (an optimised script is available at GITHUB URL). To use this option, create a sheet with the class list: Name, last name, and email address (as below).
  4. You can test the addon by completing the inputs in the 'Noisy Sheets' addon. Using the Google Forms option, an "information" sheet should be created; you can test the addon is working by completing the google form, which should automatically create a copy and set of answers. We recommend testing the class list with a small number to begin with.
  5. All copies of the spreadsheet will appear in the same folder as your original data. The Answers for each individual will be collated in a new sheet called MasterAnswers.
